# 영업실 결산 대시보드 (신규 사이트)

기존 `26Y_SC_영업 및 소싱 현황표.xlsx`의 **ALL데이터** 시트만 업로드하면, 나머지 집계표(영업 지표 → 추후 매니저별/셀러별/상품별 등)를 수식 없이 자동으로 계산해서 보여주는 사이트입니다.

이 사이트는 결산/내부 자료용이니, 링크는 필요한 인원에게만 공유해주세요.

## 실행 방법 (Windows, 더블클릭)

1. Python이 없다면 먼저 설치: https://www.python.org/downloads/ (설치 시 "Add python.exe to PATH" 체크)
2. 이 폴더(`sc_dashboard`) 전체를 다운로드
3. 폴더 안의 **`실행.bat`**을 더블클릭
   - 처음 실행 시 몇 분 정도 걸릴 수 있어요.
   - 검은 터미널 창이 함께 열리는데, 앱이 돌아가는 중이라는 뜻이니 그대로 두세요. 닫으면 앱이 종료됩니다.

## 사용 방법

1. 첫 화면에서 "영업 지표" 메뉴로 들어갑니다.
2. `26Y_SC_영업 및 소싱 현황표.xlsx` 파일(ALL데이터 시트 포함)을 업로드합니다.
3. 월별 진행 건수 / 매출 결과 / GP 결과 표와 그래프가 자동으로 나오고, 엑셀로도 다운로드할 수 있습니다.

## 집계 로직 (기존 엑셀 수식 기준으로 검증 완료)

- **매출 결과** = 해당 월 전체 행의 '매출' 합계
- **GP 결과** = 해당 월 전체 행의 '트리즈GP' 합계
- **진행 건수** = 해당 월 전체 행 수에서 '다이렉트/벤더'가 **기타** 또는 **PA**인 행은 제외한 개수
  (기존 시트의 "글로벌 소싱 + 셀러 + 벤더사" 합산 방식과 동일)

## 다음 순서

매니저별 → 셀러별/셀러상품별 → 상품별 뷰를 이어서 추가할 예정입니다.
